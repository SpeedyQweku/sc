# sc

Use to get inscope subdomain

## Installation

```bash
go install github.com/SpeedyQweku/sc@latest
```

## Usage

```bash
sc -l <domain.txt> -s <sub_recon.txt> | tee -a result.txt
```

```bash
-d string
  The target inscope domain
-l string
  The target inscope domains file
-s string
  Subdomains file
```
